#  House Price Prediction using Machine Learning

##  Objective

The goal of this project is to predict house prices based on features such as square footage, number of bedrooms, and location using regression models.

---

##  Dataset

* Source: Kaggle House Price Dataset
* Features include:

  * Square Footage
  * Bedrooms
  * Location
  * Other property-related attributes

---

##  Project Workflow

### 1. Data Preprocessing

* Removed irrelevant columns (e.g., IDs)
* Handled missing values
* Encoded categorical variables (e.g., location)
* Removed outliers in price

---

### 2. Exploratory Data Analysis (EDA)

* Visualized distributions of features
* Analyzed feature correlations
* Identified important variables affecting price

---

### 3. Model Training

Trained and compared three regression models:

* Linear Regression (baseline)
* Ridge Regression (regularization)
* Gradient Boosting (advanced model)

---

### 4. Model Evaluation

Models were evaluated using:

* MAE (Mean Absolute Error)
* RMSE (Root Mean Squared Error)
* R² Score

---

### 5. Visualizations

The following outputs were generated:

| File Name               | Description                  |
| ----------------------- | ---------------------------- |
| eda_charts.png          | Data exploration visuals     |
| actual_vs_predicted.png | Model prediction comparison  |
| residual_analysis.png   | Error distribution analysis  |
| feature_importance.png  | Key influencing features     |
| model_metrics.png       | Model performance comparison |

---

##  Results

* Gradient Boosting performed the best among all models
* Feature importance analysis highlighted key price drivers
* Residual plots confirmed model consistency
